Use the buttons at the top to set states. Use the first two dropdowns to identify the batch of states you want to use (first is the test you're doing, second is the specific batch within that test).
Once you've selected a state, to set it click "Send Command"

You can also enable/disable each of the four graphs by checking/unchecking the box underneath. To hide a specific chart element, click it in the legend and the plot will automatically rescale.

For additional test, you should add new states into the STATE_SETS.json file (in resources)
